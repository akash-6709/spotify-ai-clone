# 🎵 Spotify AI Clone

A full-stack Spotify-like music player with AI-powered song recommendations using Google Gemini API.

## 🚀 Features
- Play/pause songs, next/previous track controls
- Songs served from custom Node.js backend
- AI-powered song recommendations
- RESTful API architecture

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Express
- **AI:** Google Gemini API

## 🔧 How to Run

### Backend
```bash
cd backend
npm install
node server.js
